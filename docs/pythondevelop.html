Product Design Specifications

To be considered a success, my application must meet the following:

Target Audience

1) The target audience of this application are people who want a easy way to talk to people online without all their data being collected.

User Experience

1) Very user-friendly and the user should not have any problems regarding use of the application.
2) The interface should look good and be very simple. There should not be any clicky-ness and all the widgets / components should not be used in a way that distracts from the main point
3) Every instance of the app should show the same thing and be consistent every time.
4) There will be lots of testing with many users for lots of feedback to constantly improve the product.

User Interface Elements

1) The color scheme should remain consistent through the application.
2)Typography will not be all over the place and keep a consistent design.
3)Images / White space, there will not be too many images and there will have to be some white space to make the application look a lot better.

Center Window

def center(win):
win.update_idletasks()
width = win.winfo_width()
height = win.winfo_height()
x = (win.winfo_screenwidth() // 2) - (width // 2)
y = (win.winfo_screenheight() // 2) - (height // 2)
win.geometry('{}x{}+{}+{}'.format(width, height, x, y))
