Target UX UI Material Design

Target Audience

The target audience of this application will be people who don't already know about this issue. When the target audienc finds out, it is possible that things will be done about it. Next, there is: UX

UX

UX or User Experience is how the user interacts with the page. It is also how easy the product is to use. This means that I would want my application to be as user-friendly as possible with multiple spots for 'how to use' and just a clean layout with nothing disruptive that may make the viewer dislike the page in any way.

UI

UI or User interface is essentially the visuals only that the page will have. Things I will focus on include:

  • Color scheme (white / black / orange)
    A consistent color scheme is always very helpful because it allows the colors to match and pop when they need to.
  • White Space
    I don't want to make everything too crowded because that will look bad asthetically.
  • Typography
    I'm going to stick to only a few types of fonts and use them where they should. I will choose Roboto because it's clean and popular right now.
  • Images
    Lack of images will result in a lot of text. I want lots of images and visuals to allow the user to get information without being bored.

Material Design

This is something I want to mention because of how modern it is. It's a design language which makes everything very intuitive and easy to use and helps with the overall theme of the app. It removes all clutter and has very nice alignments for everything. I want to incorperate this into my app because I think just the baisc design language of an app can thoroughly change the outcome from a historic layout to a very modern / futuristic one and that is why I want to include it.

Success Critera

What I need to do in order to create the solution.




I will know my solution is a success when...


  • Visuals
  • Graphics
  • Use



- Everything on the page is marginalized correctly.

- Everything follows similar design standards with the same color scheme consistent.

- There is a balance of typography, elements and white space.

- Everything on the page has a purpose and nothing is distracting.

- The entire visual aspect comes together as one to create a unifying and eye-opening experience.

- The visuals do not limit the impact the issue has on people




- The graphs are helpful and informing.

- The graphs are interactive and you are able to customize what is on the graph while outputting accurate data.

- The graphs are connected seamlessly.

- There is a balance of images, text and graphs.

- The images do not clash with the color scheme and theme.

- There are graphics such as arrows and guides only when neccessary.

- The page is not crowded with graphics.




!-! The entire solution is easy to use and can be used by ANYONE.

- The entire solution can be customized to the user's liking.

- It is easy to input your own countries / year / gho into the graphs.

- 5/5 people in the class can use the website with no problems.

- The user can get a good web experience from it while having being impacted by the issue.

- The use of the website does not detract from the impact the issue has on people.
